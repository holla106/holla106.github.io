第一篇文章就說一下這個 blog 是怎樣做出來的

這個 blog 是使用 hugo 來建置的，主題是使用 neopost 再加以修改而成
（我其實挺喜歡這種懷舊風主題，有著小時候逛的網站的感覺）

感覺好像可以打一篇短短的教學文（？

以下是步驟：

1. 安裝 Hugo：

  • macOS: brew install hugo
  • ArchLinux: pacman -S hugo
  • Windows: winget install Hugo.Hugo

2. 初始化專案：

hugo new site my-blog
cd my-blog

這會生成一個基本的 Hugo 專案結構：

- my-blog/
├── archetypes/    # 內容模板
├── content/       # 存放 Markdown 內容
├── layouts/       # 自訂模板
├── static/        # 靜態資源（如圖片、CSS）
├── themes/        # 主題資料夾
├── hugo.toml    # 配置文件

3. 下載主題：（以neopost為例）

git clone "https://github.com/salatine/neopost.git" themes/neopost

參閱主題的其他說明
